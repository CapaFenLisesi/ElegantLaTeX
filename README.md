## How to use
* `\documentclass[]{elegant}`
* XeLaTeX, pdfLaTeX, LaTeX 编译均可

## What have done
* 选项
    * 默认的选项`utf`,`withchinese`,`withisomath`
    * 通过`utf`, `gbk`来调整编码
    * 通过`dvips`, `dvipdfm`, `pdftex`, `xetex`来调整代码适应
    * 通过`withchinese`/`withoutchinese`来开关中文支持
    * 通过`withisomath`/`withoutisomath`来开关iso数学支持
* 检测编译方式
* 中文支持
* iso数学支持
* 交叉引用
* 字号支持
